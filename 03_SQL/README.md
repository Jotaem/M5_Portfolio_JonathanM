## 03_SQL: Consultas y Lógica de Negocio

Esta sección incluye consultas complejas para obtener información valiosa y el manejo de transacciones.

* `01_consultas_basicas.sql`: Consultas SELECT simples y con funciones de agrupación (SUM, COUNT).
* `02_joins_subqueries.sql`: Consultas avanzadas con INNER/LEFT JOINs y Subconsultas (ej. productos no vendidos en el último mes).
* `03_transaccion_compra.sql`: Implementación de una Transacción Atómica utilizando un Stored Procedure con START TRANSACTION, COMMIT y ROLLBACK.
