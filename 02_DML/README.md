## 02_DML: Manipulación de Datos

Esta carpeta contiene el código para modificar los datos dentro de las tablas.

* `01_insert_data.sql`: Sentencias INSERT INTO para poblar las tablas `productos`, `proveedores` y `transacciones`.
* `02_update_delete.sql`: Demuestra operaciones UPDATE (ej. actualizar stock después de una venta) y DELETE, aplicando primero la eliminación en la tabla de transacciones para mantener la integridad referencial.
